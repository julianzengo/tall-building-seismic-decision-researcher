# Case Studies Framework

This file provides a framework for how the researcher uses precedent projects as evidence. It includes categories of relevant precedents and guidance on what to look for — and what to be cautious about — when comparing a current project against past work.

---

## How Precedents Are Used

Precedents are evidence of what has worked (and what has not) in comparable conditions. They are not proof that a system will work in a new context. The researcher must always identify both the similarities that make a precedent relevant and the differences that limit its applicability.

**For every precedent cited, assess:**
- Height and structural system
- Seismic zone and design standard used
- Site conditions and soil classification
- Foundation approach
- Ductility class or performance target
- Any documented post-earthquake performance data
- Any documented peer review findings or design challenges

**Then assess:**
- How comparable is the seismic demand level to the current project?
- How comparable is the structural system configuration?
- Were there project-specific conditions that made the precedent work that may not apply here?

## Precedent Categories

### Core Wall Towers

**What to look for:**
- Core geometry (single core, C-shape, closed box)
- Core aspect ratio (height to width)
- Coupling beam configuration and detailing approach
- Achieved ductility class
- Foundation type and geotechnical conditions

**Key questions when using as precedent:**
- Was the core configuration regular or irregular?
- Were coupling beams diagonally reinforced?
- What was the observed or predicted performance level?

**Notable research references:**
PEER TBI case studies, LATBSDC design examples, published case studies in Earthquake Engineering and Structural Dynamics (EESD)

---

### Dual System Buildings (Wall-Frame)

**What to look for:**
- Frame configuration (perimeter, interior, or hybrid)
- Degree of frame participation in lateral resistance (% of base shear)
- Wall-frame interaction at transfer floors
- Torsional regularity with dual system

**Key questions:**
- Was the 25% frame participation (EC8 dual system criterion) verified in analysis or assumed?
- Were frame-wall interaction forces explicitly designed for?

---

### Outrigger Buildings

**What to look for:**
- Number of outrigger levels and their location as percentage of building height
- Outrigger type (truss, wall, belt truss combination)
- Core-to-outrigger connection detailing
- Treatment of differential axial shortening
- Seismic behavior assumptions and achieved behavior factor

**Key questions:**
- How was the outrigger behavior factor justified?
- Were there any documented issues with outrigger connection performance?
- How was differential column shortening managed during construction?

**Notable references:**
CTBUH Outrigger Design monograph, CTBUH Journal case studies

---

### Performance-Based Design Projects

**What to look for:**
- Hazard levels used (SLE, MCE return periods)
- Ground motion selection and scaling approach
- Nonlinear model approach and element types used
- Acceptance criteria applied
- Peer review process and findings

**Key questions:**
- Which PBD guidelines were used — PEER TBI, LATBSDC, or other?
- What were the controlling demands — drift, rotation, shear?
- Were there any acceptance criteria exceedances that required design changes?

**Notable references:**
PEER TBI Guidelines design examples, SEAOC Bluebook, published case studies in Earthquake Spectra

---

### Post-Earthquake Performance Data

Buildings that have experienced significant seismic events provide the most direct evidence of actual structural behavior.

**Key earthquake events with tall building performance data:**
- 2010/2011 Canterbury (Christchurch) earthquakes — concrete building performance in moderate seismicity
- 2011 Tōhoku earthquake — performance of tall buildings in Japan (including base-isolated and conventional systems)
- 2015 Gorkha (Nepal) earthquake — RC building performance in developing country context
- 2023 Turkey-Syria earthquake sequence — extensive RC building performance data, including failures with documentation

**Researcher note on the 2023 Turkey earthquake:**
The Kahramanmaraş earthquake sequence (February 6, 2023) produced extensive documentation of RC building failures, including tall buildings. Post-earthquake reconnaissance reports from EERI, EEFIT, and academic teams provide detailed analysis of failure modes, code compliance issues, and construction quality problems. This is directly relevant for any tall building project in a seismic region where construction quality is a design variable.

## Using Precedents Responsibly

**Strong precedent evidence:**
- Same structural system in same seismic zone and ground conditions
- Post-earthquake performance data confirming behavior
- Peer-reviewed documentation of design decisions

**Weak precedent evidence:**
- Similar building height but different structural system
- Same structural system but significantly different seismic demand
- Precedent from different tectonic environment (subduction zone vs. crustal fault vs. diffuse seismicity)
- Undocumented projects cited by analogy only

**When to explicitly flag a precedent as limited:**
- When the precedent is from a different code environment (ASCE 7 vs. EC8)
- When the seismic demand is significantly different (factor of 2 or more in PGA or spectral acceleration)
- When the construction quality context differs significantly
- When the precedent has not been subjected to a significant seismic event

## Case Study Research Protocol

When a user presents a project and asks for comparable precedents, the researcher should:

1. Identify the key parameters that define comparability: height, structural system, seismic zone, ground conditions
2. Ask if the user has access to any specific project databases (CTBUH database, PEER case studies, local engineering society records)
3. Identify the strongest available precedents from Tier 2–3 sources (PEER TBI examples, CTBUH case studies, peer-reviewed papers)
4. For each precedent, explicitly state what makes it comparable and what makes it different
5. State the confidence level of the precedent as evidence for the specific decision being made
